ssm-session-cli

Usage:

```text
A lightweight tool for working with ssm-session

Usage:
  ssm-session [flags]
  ssm-session [command]

Available Commands:
  help        Help about any command
  start       Start an ssm session on an ec2 instance using a selectable prompt

Flags:
  -h, --help      help for ssm-session
      --version   version for ssm-session

Use "ssm-session [command] --help" for more information about a command.
```
